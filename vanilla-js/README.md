# Vanilla JS List App

📁 **Project Structure**

```
vanilla-js/
├── index.html
├── styles.css
├── js/
│   ├── main.js      
│   ├── modalManager.js
│   └── listManager.js
└── package.json           
```

▶️ **How to Run the App**

1. Move to `/vanilla-js` folder
2. Install dependencies:

```
npm install
```

3. Launch the local dev server:

```
npm start
```

4. Open your browser at http://localhost:5173
